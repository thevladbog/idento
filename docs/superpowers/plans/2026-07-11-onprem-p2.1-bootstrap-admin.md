# On-Prem Phase 2 — P2.1 Bootstrap Flow Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** On a fresh on-prem install (`DEPLOYMENT_MODE=onprem`, empty database), automatically provision one organization and one admin user from `IDENTO_ADMIN_EMAIL`/`IDENTO_ADMIN_PASSWORD` env vars, so the operator reaches a working login with no manual SQL — permanently inert once any user exists.

**Architecture:** A new `HasAnyUsers` store method backs the idempotency check. A new `backend/internal/bootstrap` package holds one function, `OnPremAdmin`, that composes `HasAnyUsers` + the already-existing, already-transactional `store.Store.ProvisionTenantWithAdmin` (added in P1.10) — no new SQL beyond the one-line existence check. `main.go` calls it once at startup, only in `onprem` mode, immediately after `EnsureSeedData`, and treats any error as fatal.

**Tech Stack:** Go 1.25 / pgx v5 (backend only — no frontend, no migration, no new table). Test tooling: `testing` + `pgxmock/v4` (store-level test, matches `pg_store_register_test.go`'s existing pattern) and a small hand-written fake implementing `store.Store` (matches `handler/testsupport_test.go`'s `fakeStore` pattern) for the bootstrap package's own tests.

## Global Constraints

- **Reuse `ProvisionTenantWithAdmin` unmodified.** It already exists at `backend/internal/store/pg_store.go:202` and already does tenant + subscription-to-default-plan + admin-user + membership in one transaction. This plan adds a caller, not new provisioning logic.
- **Idempotency gate is "any user exists," via `SELECT EXISTS(SELECT 1 FROM users)`** — not "any tenant exists," not a separate "bootstrap ran" flag/table.
- **Missing or empty `IDENTO_ADMIN_EMAIL`/`IDENTO_ADMIN_PASSWORD` on a confirmed-empty database is a fatal startup error** (`log.Fatalf` in `main.go`), not a skip and not a retry-later state. This was the user's explicit, confirmed choice.
- **`saas` mode never calls the bootstrap function.** The admin env vars stay fully inert there, exactly as today.
- **No new validation added to `config.Load()`.** `AdminEmail`/`AdminPassword`/the new `AdminOrgName` stay unvalidated at config-load time — validation (env vars required) happens only inside the bootstrap function, only when the database is confirmed empty, only in `onprem` mode.
- **New optional env var `IDENTO_ORG_NAME`**, default `"My Organization"` when unset or empty. No other new config surface.
- **No password policy.** Bootstrap accepts whatever `Register`'s existing validation already accepts (non-empty only) — do not add strength requirements.
- **Backend gate:** every task ends with `cd backend && go build ./... && go vet ./... && go test ./...` passing.

```go
// every tenant — used to detect a genuinely fresh on-prem install.
func (s *PGStore) HasAnyUsers(ctx context.Context) (bool, error) {
	var exists bool
	err := s.db.QueryRow(ctx, `SELECT EXISTS(SELECT 1 FROM users)`).Scan(&exists)
	return exists, err
}
```

```go
// Package bootstrap provisions the first organization and admin user on a
// fresh on-prem install. It is never invoked in saas mode.
package bootstrap

import (
	"context"
	"fmt"
	"log"

	"idento/backend/internal/config"
	"idento/backend/internal/store"
)

const defaultOrgName = "My Organization"

// OnPremAdmin provisions one organization and one admin user from
// cfg.AdminEmail/AdminPassword when the database has no users yet. It is
// idempotent: once any user exists (from this call or any other path),
// every subsequent call is a no-op. Callers should treat a non-nil error as
// fatal — this only runs at startup, in onprem mode, before the server
// accepts traffic.
func OnPremAdmin(ctx context.Context, s store.Store, cfg *config.Config) error {
	hasUsers, err := s.HasAnyUsers(ctx)
	if err != nil {
		return fmt.Errorf("check for existing users: %w", err)
	}
	if hasUsers {
		return nil
	}
	if cfg.AdminEmail == "" || cfg.AdminPassword == "" {
		return fmt.Errorf("first run with an empty database requires IDENTO_ADMIN_EMAIL and IDENTO_ADMIN_PASSWORD to be set — see INSTALL.md")
	}
	orgName := cfg.AdminOrgName
	if orgName == "" {
		orgName = defaultOrgName
	}
	if _, _, err := s.ProvisionTenantWithAdmin(ctx, orgName, cfg.AdminEmail, cfg.AdminPassword); err != nil {
		return fmt.Errorf("bootstrap admin: %w", err)
	}
	log.Printf("bootstrap: created organization %q with admin %s", orgName, cfg.AdminEmail)
	return nil
}
```
