# P0.3 — openapi truth-up, generated panel client, CI/Docker — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `backend/openapi.yaml` the enforceable, contract-tested truth for every panel-consumed endpoint, generate the panel's typed API client from it, and ship a CI-verified panel Docker image published at release.

**Architecture:** Backend-first: rewrite openapi.yaml group by group, each group landing together with kin-openapi contract tests that validate real handler responses in the existing httptest harness (fakeStore + newAuthedContext). A coverage ledger in TestMain (gated by `OPENAPI_COVERAGE=1`) guarantees every documented operation has a validated test. Once the spec is true, `openapi-typescript` generates a committed `schema.d.ts`, `openapi-fetch` becomes the panel's runtime client (middleware preserves the P0.2 `ApiError`/QueryCache contract), and CI gains a drift check + Docker build; release publishes `ghcr.io/<repo>-panel`.

**Tech Stack:** Go 1.25 / echo v4.15 / kin-openapi (new); openapi-typescript ^7 + openapi-fetch (new, panel); GitHub Actions; nginx-unprivileged Docker image.

**Spec:** `docs/superpowers/specs/2026-07-14-panel-p0.3-openapi-client-ci-docker-design.md` — read §3 (endpoint inventory) and §2 (decisions) before starting.

## Global Constraints

- Branch: `panel/p0.3-openapi-ci-docker` (exists; spec committed).
- The Go handlers are the source of truth in P0.3. On any spec↔code mismatch found by a contract test, **fix the spec**, never loosen validation or change handler behavior.
- Preserve the real path asymmetry: `POST /auth/login`, `POST /auth/register`, `POST /auth/login-qr` live under `/auth/*`; everything else under `/api/*`. The old `/api/auth/login` paths in the current openapi.yaml are wrong and must be deleted.
- Two error shapes exist and both must be documented honestly: handlers that do `c.JSON(status, map[string]string{"error": ...})` → component `Error` (`{error, code?}`); handlers that `return echo.NewHTTPError(...)` → echo renders `{"message": ...}` → component `HTTPError`. Never document `{error}` for an endpoint that actually returns `{message}` — the contract test will catch it.
- Out-of-scope surfaces stay undocumented and are listed in the openapi.yaml header comment: `/api/mobile/*`, `POST /api/zones/{zone_id}/scan`, `POST /api/events/{event_id}/checkins/batch`, `GET|POST /api/sync`, `POST /api/public/import`, `/api/super-admin/*`, `GET /api/version`, `POST /api/util/printers/generate-qr`, `GET /openapi.yaml`, `GET /docs`, `GET /printer-qr`.
- openapi.yaml is embedded into the backend binary (`backendOpenAPISpec` in `main.go`, served at `/openapi.yaml` + Scalar UI at `/docs`) — the file must stay at `backend/openapi.yaml`.
- Do not pin exact npm/Go dependency minor versions in code you write from this plan — install latest within the stated major (`openapi-typescript@^7`, `openapi-fetch` latest, `kin-openapi` latest) and commit what the resolver picked. (P0.2 lesson: a guessed patch version didn't exist.)
- Panel client migration must NOT change: `ApiError` class, `authErrorKey`, session module, screen call signatures (`login(email, password)` etc.), or the QueryCache/MutationCache global handlers in `panel/src/app/queryClient.ts`.
- Every npm workspace already inherits root `.npmrc` (`registry=https://registry.npmjs.com/`) — do not add per-package .npmrc.
- OpenAPI version: `3.0.3` (kin-openapi and openapi-typescript both fully support it; do not use 3.1).
- Commit after every green step; run `gofmt` on Go files before committing.

## Conventions (referenced by Tasks 1–8)

**YAML conventions** — established in Task 1, applied identically in every group task:

- Path params: OpenAPI `{param}` syntax (echo `:param` in Go). Param schemas: `type: string, format: uuid` for IDs.
- Success shapes come from the actual Go handler return (`c.JSON(status, X)`): named struct → component named after the struct; anonymous `map[string]interface{}` → inline object or purpose-named component (e.g. `AuthResponse`).
- Model components mirror JSON tags exactly, including `json:"-"` omissions and `omitempty` (omitempty ⇒ not in `required`). Timestamps: `type: string, format: date-time`. Nullable pointers that serialize as `null`: `nullable: true`.
- Every operation documents every status its handler can actually produce (read every `return c.JSON(...)` / `echo.NewHTTPError` branch), plus `429` (HTTPError) on the four rate-limited routes (`/auth/login`, `/auth/login-qr`, `/api/stations/provision`, `/api/zones/checkin`).
- All `/api/*` protected operations get `security: [{bearerAuth: []}]` and a `403` response with the `Error` schema (TenantGate can return `{code: "tenant_suspended", error: ...}` on any of them). Public endpoints: `security: []`.
- Non-JSON responses: document the real content type (from `c.Blob`/CSV writer in the handler): CSV export `text/csv`, QR images `image/png`, badge ZPL `text/plain`, font CSS `text/css`, font file — whatever `GetFontFile` passes to `c.Blob`. Schema: `type: string, format: binary` (or plain `string` for text).
- `operationId` on every operation: lowerCamelCase of the handler method (e.g. `login`, `getEvents`, `createAPIKey`).

**Contract-test conventions:**

- New contract tests live in per-group files `backend/internal/handler/openapi_contract_<group>_test.go`, package `handler`.
- Use the existing harness: `fakeStore` (override only needed fields), `newAuthedContext`/`newUnauthedContext`, `c.SetPath` + `c.SetParamNames`/`SetParamValues` for param routes, `t.Setenv("JWT_SECRET", "test-secret")` when the handler mints tokens.
- The `url` passed to `validateResponse` is the **concrete** request URL (real UUIDs), same one passed to `newAuthedContext`.
- One happy-path contract test per operation is the floor (the coverage ledger enforces it). Where an existing test already exercises an operation, still add the contract test — do not retrofit old tests.

```go
package handler

import (
	"context"
	"fmt"
	"net/http/httptest"
	"os"
	"sort"
	"strings"
	"sync"
	"testing"

	"github.com/getkin/kin-openapi/openapi3"
	"github.com/getkin/kin-openapi/openapi3filter"
	"github.com/getkin/kin-openapi/routers"
	"github.com/getkin/kin-openapi/routers/gorillamux"
)

// The contract harness validates recorded handler responses against
// backend/openapi.yaml. The spec is the contract; handlers are the truth.
// On mismatch, fix the spec (P0.3 rule) — never silence the test.
var (
	specOnce   sync.Once
	specDoc    *openapi3.T
	specRouter routers.Router
	specErr    error

	coverageMu sync.Mutex
	// keys: "METHOD /spec/path/{template}" — written by validateResponse,
	// asserted complete by TestMain when OPENAPI_COVERAGE=1.
	coverage = map[string]bool{}
)

func loadSpec(t *testing.T) routers.Router {
	t.Helper()
	specOnce.Do(func() {
		loader := openapi3.NewLoader()
		specDoc, specErr = loader.LoadFromFile("../../openapi.yaml")
		if specErr != nil {
			return
		}
		if specErr = specDoc.Validate(loader.Context); specErr != nil {
			return
		}
		specRouter, specErr = gorillamux.NewRouter(specDoc)
	})
	if specErr != nil {
		t.Fatalf("backend/openapi.yaml failed to load or validate: %v", specErr)
	}
	return specRouter
}

// validateResponse matches the concrete request URL against openapi.yaml and
// validates rec's status/headers/body against the matched operation's schema.
// An undocumented path or status fails the test.
func validateResponse(t *testing.T, method, url string, rec *httptest.ResponseRecorder) {
	t.Helper()
	router := loadSpec(t)
	req := httptest.NewRequest(method, url, nil)
	route, pathParams, err := router.FindRoute(req)
	if err != nil {
		t.Fatalf("%s %s is not documented in openapi.yaml: %v", method, url, err)
	}
	input := &openapi3filter.RequestValidationInput{
		Request:    req,
		PathParams: pathParams,
		Route:      route,
		Options:    &openapi3filter.Options{AuthenticationFunc: openapi3filter.NoopAuthenticationFunc},
	}
	respInput := &openapi3filter.ResponseValidationInput{
		RequestValidationInput: input,
		Status:                 rec.Code,
		Header:                 rec.Header(),
	}
	respInput.SetBodyBytes(rec.Body.Bytes())
	if err := openapi3filter.ValidateResponse(context.Background(), respInput); err != nil {
		t.Errorf("%s %s → %d does not match openapi.yaml:\n%v", method, url, rec.Code, err)
	}
	coverageMu.Lock()
	coverage[method+" "+route.Path] = true
	coverageMu.Unlock()
}

func TestMain(m *testing.M) {
	code := m.Run()
	if code == 0 && os.Getenv("OPENAPI_COVERAGE") == "1" {
		if err := assertSpecCoverage(); err != nil {
			fmt.Fprintln(os.Stderr, err)
			code = 1
		}
	}
	os.Exit(code)
}

// assertSpecCoverage fails when openapi.yaml documents an operation no
// contract test validated. Gated behind OPENAPI_COVERAGE=1 so a local
// `go test -run TestX` on a single test doesn't spuriously fail.
func assertSpecCoverage() error {
	if specDoc == nil {
		loader := openapi3.NewLoader()
		doc, err := loader.LoadFromFile("../../openapi.yaml")
		if err != nil {
			return fmt.Errorf("coverage check could not load openapi.yaml: %w", err)
		}
		specDoc = doc
	}
	var missing []string
	for path, item := range specDoc.Paths.Map() {
		for method := range item.Operations() {
			key := method + " " + path
			if !coverage[key] {
				missing = append(missing, key)
			}
		}
	}
	if len(missing) == 0 {
		return nil
	}
	sort.Strings(missing)
	return fmt.Errorf("openapi.yaml operations with no contract test (add one calling validateResponse):\n  %s",
		strings.Join(missing, "\n  "))
}
```

package handler

import (
	"net/http"

	"github.com/labstack/echo/v4"

// RegisterMetaRoutes mounts the public health and instance-metadata routes.
// They live in main.go historically; extracted so they are unit-testable and
// contract-testable like every other handler.
func RegisterMetaRoutes(e *echo.Echo, mode, version string) {
	e.GET("/health", Health)
	e.GET("/api/instance", Instance(mode, version))
}

// Health reports process liveness.
func Health(c echo.Context) error {
	return c.JSON(http.StatusOK, map[string]string{"status": "ok"})

// Instance returns deployment metadata the frontends read before login.

	return func(c echo.Context) error {
		return c.JSON(http.StatusOK, map[string]interface{}{
			"mode":    mode,
			"version": version,
			"license": nil,
		})
	}
}
